## ✨ Key Features

### 🎯 Card Management
- **Add, Edit, Delete** cards with ease
- **Categorize cards** by owners and types
- **Copy and Share** cards with details
- **Duplicate detection** and prevention within same owner

### 📊 Statistics
- Detailed card statistics display
- Used and unused card counts
- Copy and share operation tracking
- Cards grouped by owners and types

### 📁 Import & Export
- **PDF Import** using PDF.js
- **JSON Import** and **CSV Import**
- **Export to JSON** and **CSV**
- **Data backup** capabilities

### 🌙 User Interface
- **Dark mode** for eye comfort
- **Responsive design** for all screens
- **Full Arabic support** (RTL layout)
- **Clean and simple** interface

### 📱 Android Support
- Full **WebView** compatibility
- **Share** and **Save** files via Android
- **Open links** from within the app

## 🚀 How to Use

### 1️⃣ Add a New Card
1. Click the **"Add"** button in the table
2. Enter the card number, select type and owner
3. Click **"Save"**

### 2️⃣ Import Files
- **PDF**: Click the **"PDF"** button and select a PDF file
- **JSON/CSV**: From the dropdown menu select **"Import File"**

### 3️⃣ Export Data
- Click **"Export"** to export to JSON or CSV

### 4️⃣ Copy & Share
- Use the **📋 Copy** and **📤 Share** icons next to each card

### 5️⃣ Statistics
- Navigate to the **"Statistics"** tab for detailed reports
